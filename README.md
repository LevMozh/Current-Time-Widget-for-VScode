**Группа:** М3105

## Описание

Плагин для VS Code, который отображает текущее время в статус-баре редактора. Время автоматически обновляется каждую секунду и показывает актуальное время в формате ЧЧ:ММ:СС. При клике на виджет отображается полная текущая дата.

## Возможности

- **Отображение времени**: текущее время в формате ЧЧ:ММ:СС в статус-баре
- **Автообновление**: время обновляется каждую секунду в реальном времени
- **Интерактивность**: при клике на время отображается полная дата
- **Автозапуск**: плагин автоматически активируется при запуске VS Code
- **Корректное завершение**: ресурсы освобождаются при деактивации плагина

## Установка

### Для использования
1. Скачайте .vsix файл из релизов
2. В VS Code: Ctrl+Shift+P → "Extensions: Install from VSIX"
3. Выберите скачанный файл

```bash
git clone https://github.com/LevMozh/Current-Time-Widget-for-VScode.git
