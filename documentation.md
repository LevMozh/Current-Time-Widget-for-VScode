# Документация расширения Current Time Widget для VS Code

## Описание

**Current Time Widget** — расширение для Visual Studio Code, которое отображает текущее время в статус-баре редактора. Плагин автоматически обновляет время каждую секунду и предоставляет дополнительную информацию о дате при клике на виджет.

---

## Основные функции и возможности

- Отображение текущего времени в формате ЧЧ:ММ:СС в статус-баре VS Code
- Автоматическое обновление времени каждую секунду
- Показ полной текущей даты при клике на виджет времени
- Автозапуск при старте VS Code
- Корректное освобождение ресурсов при деактивации расширения

---

## Сценарий работы

1. Пользователь открывает VS Code
2. Расширение автоматически активируется
3. В правом нижнем углу статус-бара появляется текущее время
4. Время обновляется каждую секунду в реальном времени
5. При клике на время отображается всплывающее сообщение с полной датой
6. При закрытии VS Code расширение корректно деактивируется

---

## Документация функций

### Функция getCurrentTime

- Получает текущее время в формате ЧЧ:ММ:СС

**Параметры:**
- отсутствуют
 
**Возвращаемое значение:**
- string: отформатированная строка времени

**Пример вызова:**
```
getCurrentTime()
"14:30:25"
```

### Функция getCurrentDate

- Получает текущую дату в полном текстовом формате

**Параметры:**
- Отсутствуют

**Возвращаемое значение:**
- string: отформатированная строка даты

**Пример вызова:**
```
getCurrentDate()
"понедельник, 25 декабря 2023 года"
```

### Функция activate
- Активирует расширение при запуске VS Code

**Параметры:**
- context (vscode.ExtensionContext): контекст расширения VS Code

**Возвращаемое значение:**
- void: функция не возвращает значения

**Пример вызова:**
 ```
activate(context)
// Создает элемент статус-бара и запускает таймер
```

### Функция deactivate
- Деактивирует расширение при закрытии VS Code

**Параметры:**
- Отсутствуют

**Возвращаемое значение:**
- void: функция не возвращает значения

**Пример вызова:**
```
deactivate()
// Останавливает таймер и освобождает ресурсы
```
